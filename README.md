## Neovim
Contains the files for customization.   
It includes the Plug installer, `coc-install.json`, and `init.vim` files.   
To install:
* Install `neovim`.
* Copy the nvim folder in `%AppData%/../Local` folder.
* Execute this command to install the plugins.
```
nvim +PlugInstall
```
